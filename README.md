## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/gofinances.git
```

```
$ cd gofinances
```

Install all the required dependencies

```
$ yarn
```

Runs the [api](https://github.com/HarrisonHenri/finapi)

Then execute

```
$ yarn start
```
